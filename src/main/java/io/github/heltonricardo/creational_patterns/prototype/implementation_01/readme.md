#### Usando shallow copy (cópia rasa):

Supondo a estrutura:

- ObjetoOriginal

  - campo1: int / Integer
  - campo2: Referencia

- ObjetoCopia
  - campo1: int / Integer
  - campo2: Referencia

As referências do ObjetoCopia são as mesmas do ObjetoOriginal, ou seja,
"apontam" para o mesmo local de memória, para o mesmo objeto Referencia.
